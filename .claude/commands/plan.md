# Plan Feature

Create a design document for the requested feature:

1. **Goal**: What problem does this solve?
2. **Scope**: What's in/out for this iteration?
3. **Design**: Data flow, module boundaries, key interfaces
4. **Models**: New/modified Pydantic models (in tello-core if cross-service)
5. **Testing strategy**: What to test and how
6. **Open questions**: Anything that needs human input

Output: Save to `docs/plans/YYYY-MM-DD-feature-name.md`
Do NOT write any implementation code.
